# Contributing to Cryptocurrency Price Predictor

We welcome contributions from the community! Here's how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/crypto-price-predictor.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Commit: `git commit -m 'Add your feature'`
6. Push: `git push origin feature/your-feature`
7. Open a Pull Request

## Code Style

We follow PEP 8 with some modifications:
- Use 4 spaces for indentation
- Max line length: 100 characters
- Use type hints where possible
- Write docstrings for all functions

## Testing

Before submitting a PR:
1. Run existing tests
2. Add new tests for your changes
3. Ensure all tests pass

## Reporting Issues

When reporting issues, please include:
1. Python version
2. OS and environment
3. Steps to reproduce
4. Expected vs actual behavior
5. Log files if applicable

## Feature Requests

Feature requests are welcome! Please include:
1. Use case description
2. Expected behavior
3. Any alternatives you've considered

## Questions?

Feel free to:
- Open an issue for discussions
- Join our Discord community
- Email maintainers
